Como usar no Gemini
01
Anexe a imagem
No Gemini, clique no ícone de imagem e envie sua foto
02
Cole o prompt
Copie o prompt do estilo desejado e cole na caixa de texto
03
Ajuste [subject]
Se o prompt tiver [subject], substitua pelo seu tema específico
04
Itere
Adicione "more dramatic" ou "less saturated" para refinar o resultado
05
Preserve
Adicione "preserve original composition and subject" para manter a estrutura
06
Negatize
Adicione "do not add [x]" para evitar elementos indesejados
